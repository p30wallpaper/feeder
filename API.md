# Unnamed Feed Reader API Specification

## List all feeds
GET /feeds/

## Add feed
POST /feeds/

## Get feed items and details
GET /feeds/:feed\_id

## Update feed details
PATCH /feeds/:feed\_id

## Delete feed
DELETE /feeds/:feed\_id

## Get feed item
GET /feeds/:feed\_id/:item\_id

## Update item details
PATCH /feeds/:feed\_id/:item\_id
